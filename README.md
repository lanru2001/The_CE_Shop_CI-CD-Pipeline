# aws-cicd-pipeline-project

A simple ReactJS app that displays latest stats about COVID-19.

AWS CI/CD Pipeline using Github, CodeBuild, CodeDeploy and CodePipeline that automate the entire build and deployment process after a new commit.

The CI/CD CodePipeline architecture use the tools , platforms and service :

ElasticCompute service for apache deployment server 

AWS Identity and Access Management - roles and policies 

Simple storage service - storing artifacts 

Cloudwatch - for monitoring , alert and logging 

GitHub - code repository

CodeBuild - for building the code 

codeDeploy - for deploying the application

CodePipeline - for creating the CI/CD Pipeline
